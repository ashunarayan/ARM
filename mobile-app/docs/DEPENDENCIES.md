# Package.json Updates Required

Add these dependencies to your `package.json`:

```json
{
  "dependencies": {
    "@tensorflow/tfjs": "^4.22.0",
    "@tensorflow/tfjs-react-native": "^0.8.0",
    "expo-asset": "~11.0.3",
    "expo-file-system": "~19.0.1",
    "expo-gl": "~15.0.3",
    "expo-sensors": "~15.0.5"
  }
}
```

## Installation Command

```bash
npm install @tensorflow/tfjs@^4.22.0 @tensorflow/tfjs-react-native@^0.8.0 expo-asset@~11.0.3 expo-file-system@~19.0.1 expo-gl@~15.0.3 expo-sensors@~15.0.5
```

## Verification

After installation, verify:

```bash
npm list @tensorflow/tfjs
npm list expo-sensors
npm list expo-gl
```

All should show as installed without errors.
